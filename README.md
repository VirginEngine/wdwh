# WDWH (Easyer web dev without html and reduced css)

## Install

1. Init blank bun

```sh
bun init
```

2. Install `@virgin-engine/wdwh`

```sh
bun i @virgin-engine/wdwh
```

3. Add script init to `package.json`

```json
{
  "private": true,
  "scripts": {
    "init": "wdwh init"
  }
  // ...
}
```

4. Run init

```sh
bun run init
```

5. Install deps

```sh
bun i
```
